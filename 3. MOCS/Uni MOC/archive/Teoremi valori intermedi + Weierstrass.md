>[!def] Valori intermedi:
>- se:
>	- $\lim_{ x \to +\infty }f(x) = \pm \infty$
>	- $\lim_{ x \to -\infty }f(x) = \mp \infty$
>  
>- Allora: 
>	- $f(\mathbb{R}) = \mathbb{R}\ \ \ \textcolor{orange}{(\text{ surriettiva })}$

>[!def] Caso2:
>- Se:
>	- $\lim_{ x \to \pm\infty }f(x) = + \infty$
>- Allora: 
>	- $\exists m=min(\{f(x),\ x\in\mathbb{R}\})$
>	- $f(\mathbb{R}) = [m, +\infty]$

>[!def] Caso3:
>- Se:
>	- $\lim_{ x \to \pm\infty }f(x) = - \infty$
>- Allora: 
>	- $\exists M=min(\{f(x),\ x\in\mathbb{R}\})$
>	- $f(\mathbb{R}) = [-\infty, M]$
