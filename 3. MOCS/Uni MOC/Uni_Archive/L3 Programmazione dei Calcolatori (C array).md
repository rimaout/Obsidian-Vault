## [[C sizeof Function]]

La funzione `sizeof()` ritorna il peso in byte di un tipo di dato o variabile:

```c
int g[10];
int x = sizeof(g)
```

```c
sizeof(char));   // Output: 1
sizeof(int));    // Output: 4
sizeof(float));  // Output: 4
sizeof(double)); // Output: 8
```
