## Common commands

**Create environment:**
```shell
conda create --name ENV_NAME  
```

**Delete environment:**
```shell
conda remove --name ENV_NAME --all
```
- all: removes all the packages in the environment

**Activate environment:**
```shell
conda activate ENV_NAME
```

**Quit environment:**
```shell
conda deactivate ENV_NAME
```
